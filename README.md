
# Gothic Remake Lockpick Solver

### Use the pre-built EXE (Windows only)

1. Go to the [Releases page](https://github.com/your-username/Gothic-remake-1-auto-lockpick/releases)
2. Download the latest `GothicRemakeLockpickSolver.exe`
3. Double-click to run! No dependencies required!

### Step 1: Set up your lock

1. Select the number of plates (4-7)
2. Enter the lock name (optional)
3. Use the buttons to set each plate's position:
   - `-3` = Far left
   - `0` = Center
   - `+3` = Far right
   - Or use `W`/`S` (switch plates) and `A`/`D` (move left/right)

### Step 2: Define plate interactions

Click the buttons in the matrix to set how each plate affects the others:
- `—` = No effect
- `Same` = Moves in the same direction
- `Opp` = Moves in the opposite direction

### Step 3: Solve the lock

1. Choose a solving mode from the dropdown menu
2. Click **Solve Lock** to see the solution!

### Step 4 (Optional): Auto-solve in-game

1. Make sure you have Gothic Remake open and you're at the lockpicking screen
2. Select the **TOP plate** in the game (plate number = your selected count, e.g., 5 for a 5-plate lock)
3. Click **Auto Solve (Game)**
4. Switch to the game window within 5 seconds!
5. The app will automatically press the keys to solve the lock!
